## What is Emaaail?

It's an email service which sends user's random XKCD comic every 5 minutes

## Tech-Stack

- PHP
- HTML
- CSS
- JavaScript
- MySQL
- Google App Script

## Features

- Email verification
- Cron job scheduled which sends email to all the registered users every five minutes
