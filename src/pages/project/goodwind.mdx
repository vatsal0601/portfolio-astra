## What is GoodWind?

GoodWind is an app which is made to make their process completely online.

## Tech Stack

-   Django
-   HTML
-   CSS
-   JavaScript
-   TailwindCSS

## Features

-   Email Authentication
-   Authenticated users will see the dashboard instead of the home page, which is specifically tailored for each stakeholder (i.e. users, staff, admin)
-   Appointment scheduling
-   Document management and recommendation system (AWS)
-   Service management and recommendation system
-   Razorpay integration for payment
-   Insights (new enrollments, appointment booking, active users, etc.) for admin

_This was a paid team project in which my part was to create the frontend and some Django functionality_
